## 🚀 部署方法

### 方式一：Netlify 快速部署（1分钟上线）
直接访问 https://app.netlify.com/drop，把整个 `ai-news-website` 文件夹拖到页面上，自动获得公网访问地址。

### 方式二：Vercel 部署（推荐，免费）
1. 把项目推送到 GitHub 仓库
2. 在 Vercel 导入项目，点击部署即可自动上线

### 方式三：GitHub Pages
1. 把项目推送到 GitHub 仓库
2. 进入仓库设置 → Pages → 选择 main 分支作为源
3. 保存后自动生成访问地址：`https://<用户名>.github.io/<仓库名>/`

### 方式四：本地预览
```bash
cd ai-news-website
python3 -m http.server 8000
# 访问 http://localhost:8000 即可查看
```
